# Resource library — FMs to Virtual Cells

> *Every paper, tool, org, dataset, event, and signal we've collected on the foundation-models-to-virtual-cells story — papers we've read, models named in passing, labs to watch, money moves, competitions, perspectives. Use the filters to slice by kind, topic, status, org-type, or region.*

This is the FM-to-virtual-cells **resource library** — a landscape monitor, not a citation graph. It catalogues every paper, tool, lab, dataset, event, competition, blog, and signal we've collected, including items that are still just a name with a slide-trail attached. For the **citable subgraph** with relations and layout, see the [paper map](paper-map.md); for the methods *next door* see [adjacent methods](adjacent-methods/index.md); for the one-line model index see the [model glossary](model-glossary.md).

**How filtering works.** Each card carries faceted tags (kind / status / org / region / topic). Click any chip in the filter bar to narrow. Multiple chips in the same row are OR; rows AND together. The search box matches across name, authors, venue, and the why-it-matters line.

**Source of truth.** Everything here is generated from `data/resource-library/fm-to-virtual-cells.yaml` by `scripts/resource_library_build.py`. Add an item to the YAML, rerun the build, and it lands on the page with its tags wired up. The YAML is also the export point for [papermap](https://github.com/LiudengZhang/papermap): items with `kind: paper` and `papermap_category:` set are export-eligible.

<script>
(function() {
  const grid = document.getElementById('rl-grid');
  if (!grid) return;
  const cards = Array.from(grid.querySelectorAll('.rl-card'));
  const filters = Array.from(document.querySelectorAll('.rl-filter'));
  const search = document.getElementById('rl-search');
  const reset = document.getElementById('rl-reset');
  const counter = document.getElementById('rl-count');
  const total = cards.length;

  const active = { kind: new Set(), status: new Set(), org: new Set(), region: new Set(), topic: new Set() };

  function apply() {
    const q = (search.value || '').trim().toLowerCase();
    let shown = 0;
    cards.forEach(card => {
      const k = card.dataset.kind;
      const s = card.dataset.status;
      const o = card.dataset.org;
      const r = card.dataset.region;
      const topics = (card.dataset.topics || '').split(' ').filter(Boolean);
      const text = card.dataset.search || '';
      const passKind = !active.kind.size || active.kind.has(k);
      const passStatus = !active.status.size || active.status.has(s);
      const passOrg = !active.org.size || active.org.has(o);
      const passRegion = !active.region.size || active.region.has(r);
      const passTopic = !active.topic.size || topics.some(t => active.topic.has(t));
      const passText = !q || text.includes(q);
      const ok = passKind && passStatus && passOrg && passRegion && passTopic && passText;
      card.classList.toggle('hidden', !ok);
      if (ok) shown += 1;
    });
    counter.textContent = `Showing ${shown} of ${total}`;
  }

  filters.forEach(el => el.addEventListener('click', () => {
    const dim = el.dataset.dim;
    const val = el.dataset.val;
    const set = active[dim];
    if (!set) return;
    if (set.has(val)) { set.delete(val); el.classList.remove('active'); }
    else { set.add(val); el.classList.add('active'); }
    apply();
  }));

  search.addEventListener('input', apply);
  reset.addEventListener('click', () => {
    filters.forEach(f => f.classList.remove('active'));
    Object.values(active).forEach(s => s.clear());
    search.value = '';
    apply();
  });

  apply();
})();
</script>
